About the workshop

In algorithmic programming, we compete in coding competitions such as CodeQuest, Calico, and ACSL. These competitions focus on writing efficient code that often uses well-known algorithms to allow the code to run in a time frame. These competitions require competitors to think of solutions to a programming task and assess each solution to determine whether they are efficient enough. We focus on thinking algorithmically and learning the algorithms used in these competitions. Knowing a programming language before coming to this session is a prerequisite (this session focuses on algorithms, not how to program; please look into our CS 101 seminar if you want to learn how to program). We meet every Friday in Ms. Rachkovskiy's room (5.037) from 3:15 to 4:00.
